Status StateMachine::Job::pause ( )

Method that pauses the Job.

This Method is used to pause the Job. It can be called by StateFunctions to keep a job from being executed in the main loop and consequently shorten execution time.

Returns
Status, status_good if no error occurred.
Status StateMachine::Job::reset ( )

Method that resets the Job.

This Method is used to reset the Job. It is called by the StateMachine if the Job has produced an error. The Method should restore the Job to normal function.
